##  Linux 基础命令学习笔记

### 1. 文件与目录操作
| 命令          | 功能描述                          |
| :---------- | :---------------------------- |
| `ls`        | 列出当前目录内容          |
| `cd`        | 切换工作目录            |
| `pwd`       | 显示当前所在路径          |
| `mkdir`     | 创建新目录             |
| `rm`        | 删除文件或目录           |
| `cp` / `mv` | 复制 / 移动或重命名文件/文件夹 |

### 2. 系统信息与进程管理
| 命令 | 功能描述 |
| :--- | :--- |
| `top` |实时监控系统资源占用（CPU、内存等） ||
`ps -ef` | 查看当前系统所有运行中的进程快照  |
| `df -h` | 以易读格式查看磁盘空间使用情况  |
| `free -m` |以 MB 为单位查看内存使用情况  |

### 3. 权限管理与文本处理
| 分类 | 命令 | 功能描述 |
| :--- | :--- | :--- |
| **权限** | `chmod` | 修改文件或目录的访问权限  |
| | `chown` | 修改文件/目录的所有者  |
| | `sudo` | 以超级管理员权限执行命令 |
| **文本** | `cat` |在终端直接输出文件完整内容  |
| | `grep` | 在文本中搜索指定的字符串/模式  |
| | `vim` | 强大的终端文本编辑器  |
